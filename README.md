### Dataset Description
- File format: XLSX (Microsoft Excel)
- Language: Arabic
- Number of reviews: 292460 unbalanced / 122679 balanced
- Columns:
    hotel_id
    hotel_name
    location
    distance_from_city_center
    country
    hotel_rating
    price
    user_id
    username
    nationality
    user_rating
    review
    review_date
    traveler_type
    stay_duration

### Notes
- Files are UTF-8 encoded.
- Each row corresponds to one review.
